Les commandes de base de Git

Ce tuto permet de rappeler de maniere simple les commandes de base de Git.

Etape 1

  • Création Fichier git-1.html
  • Ajouter du code HTML base: utiliser emmet ! [return]
  • Ecrire une premiere description
  • Sauvegarder
  • Faire depuis terminal : git status => on informe l'existence nouveau fichier
  • Faire git add
  • Faire un git commit -m "Initiaisation Fichier git-1.html"

Au niveau explorateur fichier vscode, le fichier git-1.html apparait en vert, apres un add et en couleur standart apres le commit.
